Minhas Skills

CSS 3

CSS, ou Cascading Style Sheets, é uma linguagem de estilo usada para controlar a apresentação visual de documentos HTML. Ele separa o conteúdo estrutural do documento da sua apresentação, permitindo que os desenvolvedores definam o layout, cores, fontes e outros aspectos visuais de uma página web.

HTML 5

HTML, ou Hypertext Markup Language, é a linguagem utilizada para criar páginas web, definindo sua estrutura e conteúdo por meio de uma série de elementos e tags. É a base da web, permitindo a inclusão de texto, imagens, links e elementos interativos, interpretados pelos navegadores para exibir o conteúdo de forma visualmente atraente e funcional para os usuários.

Sass

Sass, ou Syntactically Awesome Stylesheets, é uma extensão da linguagem CSS que oferece recursos adicionais, como variáveis, funções e mixins, para facilitar a escrita e a manutenção de estilos em projetos web. Ele é compilado para CSS padrão antes de ser implementado.

MySQL

O MySQL é um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface. É atualmente um dos sistemas de gerenciamento de bancos de dados mais populares da Oracle Corporation.
